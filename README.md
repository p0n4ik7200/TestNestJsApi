## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

MySQL 8

```
MYSQL_ROOT_PASSWORD: my_root_password
MYSQL_DATABASE: my_database
```
